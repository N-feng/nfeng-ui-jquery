Attributes

参数 说明 类型 可选值 默认值
container 父元素 string - body
initDate 默认时间 Date - new Date()
currentToday 显示当日 boolean - true
type 类型 string week, month, year month
minDate 最小日期 string - 1990-00-00
format 格式 string - YYYY-MM-DD hh:mm:ss
locales.weekName 星期名称 array - ['日', '一', '二', '三', '四', '五', '六']
locales.monthName 月份名称 array - ['一月', '二月', '三月', '四月', '五月', '六月', '七月', '八月', '九月', '十月', '十一月', '十二月']
hideClass 隐藏类名 string - datepick-hidden
animateEnterClass 动画始名 string - slideDown-enter-active
animateLeaveClass 动画终名 string - slideDown-leave-active
