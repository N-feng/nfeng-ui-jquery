Tooltip

文字用法

Tooltip will show when mouse enter.

按钮用法

tooltip - left - small tooltip - right - small tooltip - up - large tooltip - down - large

表单用法

Element Attributes

参数 说明 类型 可选值 默认值
data-tooltip 消息文字 string - -
data-tooltip-options 方向|大小 string left/right/up/down|small/large left|small

Attributes

duration 延迟消失，单位毫秒 number - 300
offsetX 出现位置的水平偏移量 Number - 10
offsetY 出现位置的垂直偏移量 Number - 10
onClose 关闭时的回调函数, 参数为被关闭的 tooltip 实例 function - -
animateEnterClass 动画始名 string - zoom-big-fast-enter
animateEnterActiveClass 动画始激活名 string - zoom-big-fast-enter-active
animateLeaveClass 动画终名 string - zoom-big-fast-leave zoom-big-fast-leave-active
